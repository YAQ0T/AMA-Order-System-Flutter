üì± Step 2: Install Certificates

Select your device platform for installation instructions:

iOS Installation Steps:

  1. Install CA Certificate:
    • Tap the downloaded ca-cert.crt file
    • Go to Settings ‚Üí General ‚Üí VPN & Device Management
    • Tap "AMA Root CA" and install it
  2. Install Employee Certificate:
    • Tap the downloaded employee-cert.p12 file
    • Enter password: ama2024
    • Go to Settings ‚Üí General ‚Üí VPN & Device Management
    • Tap "AMA Employee Certificate" and install it
  3. Enable Trust (CRITICAL!):
    • Go to Settings ‚Üí General ‚Üí About
    • Scroll to "Certificate Trust Settings"
    • Enable trust for "AMA Root CA"

macOS Installation Steps:

  1. Double-click ca-cert.crt ‚Üí Add to System keychain
  2. Double-click employee-cert.p12 ‚Üí Enter password: ama2024
  3. Open Keychain Access ‚Üí System keychain
  4. Find "AMA Root CA" ‚Üí Double-click ‚Üí Trust ‚Üí "Always Trust"

Windows Installation Steps:

  1. Double-click ca-cert.crt ‚Üí Install Certificate ‚Üí Local Machine
  2. Place in "Trusted Root Certification Authorities"
  3. Double-click employee-cert.p12 ‚Üí Local Machine
  4. Enter password: ama2024
  5. Mark as exportable ‚Üí Automatically select store

Android Installation Steps:

  1. Go to Settings ‚Üí Security ‚Üí Encryption & credentials
  2. Install CA certificate: Tap "Install a certificate" ‚Üí "CA certificate"
  3. Select ca-cert.crt
  4. Install employee certificate: "VPN & app user certificate"
  5. Select employee-cert.p12 ‚Üí Enter password: ama2024
